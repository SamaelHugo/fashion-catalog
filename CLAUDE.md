# Fashion Catalog — ÉKOS Studio

## Project
Single-page premium fashion catalog with video hero, product grid with filtering, quick-view modal, and mini cart.
Demo project for web developer portfolio. Must look like a real high-end brand store.

## Technical Rules
- Single index.html file, all CSS and JS inline
- Vanilla HTML/CSS/JS, no frameworks
- Do NOT change CSS variables after Prompt 0
- Do NOT change fonts after Prompt 0
- Russian language for all text
- Mobile responsive (768px, 1024px)
- Images from Pexels (direct URLs via images.pexels.com)
- Video hero from Pexels Videos (direct MP4 URL)
- Product data stored in JS array (no backend)

## Design Direction
- HIGH FASHION editorial aesthetic, inspired by mosaert.com
- Clean white/off-white background with deep black typography
- Accent: warm gold/bronze (#C5A258) — used sparingly for highlights and hover states
- Video hero background — full-screen, muted, looping
- Minimal, editorial grid, lots of whitespace
- Large product photos, clean cards
- NOT colorful, NOT playful — think COS, Arket, The Row
